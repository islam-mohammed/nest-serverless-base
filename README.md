## Description

[Nest] HTTP serverless base TypeScript starter repository.

## Installation

```bash
$ npm i @vendia/serverless-express aws-lambda
$ npm i -D @types/aws-lambda serverless-offline
```

## Running the app

```bash
$ npm run build

# run locally
$ sls offline

# deploy
$ sls deply
```
